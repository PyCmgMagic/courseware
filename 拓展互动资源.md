# 拓展互动资源（需联网）

以下网页资源均已核实可用（2026-07），与第 1 课概念直接对应，适合课堂投屏演示或教师备课。
**注意：这些网站需要现场网络；课件本身完全离线可用，不依赖它们。** 建议课前打开试一遍，并把要用的页面提前加载好。

## 1. Transformer Explainer —— 最推荐，与第 1 课全程呼应

- 链接：https://poloclub.github.io/transformer-explainer/
- 演示内容：在浏览器里真实运行一个 GPT-2 小模型。输入一句英文，逐层展示 Token 切分、Embedding、注意力权重、输出概率分布，并**自带 Temperature 滑块**，拖动后实时改变候选词概率，点击生成看逐 token 接龙。
- 课堂用法：
  - 配合第 6 页（概率分布）：输入一句提示，让学生看真实的"候选词 + 概率"排行。
  - 配合第 10 页（温度滑块）：拖高温/低温各生成一次，直观比较风格差异。
  - 配合第 7 页（自回归）：点"Generate"看它一词一词往外长。
- 语言：英文界面、英文输入（正好可用"为什么中文提示要用我们自己的类比"做讨论点）。
- 网络：首次加载需下载模型文件（几十 MB），课前务必提前打开加载完成。

## 2. LLM Visualization —— 投屏效果最震撼

- 链接：https://bbycroft.net/llm
- 演示内容：把一个 GPT 模型的内部结构做成可旋转缩放的 3D 场景，矩阵、注意力连线逐层展开，可以一步一步播放一个 token 是怎样被算出来的。
- 课堂用法：第 1 课收尾或第 3 课总结时投屏 2 分钟，让学生直观感受"模型不是一句话，而是一大堆数字在流动"。适合激发兴趣，不必细讲。
- 语言：英文界面，但视觉为主，教师口头解说即可。
- 网络：加载较大，建议提前打开。

## 3. The Illustrated GPT-2 —— 教师备课图文（有简体中文版）

- 链接：https://jalammar.github.io/illustrated-gpt2/
- 内容：图解 GPT-2 的经典长文，把"预测下一个词 → 自回归 → 注意力"讲得最清楚，页面顶部可切换简体中文翻译。
- 用法：教师备课用；其中"键盘输入法就是最小的语言模型"类比可以直接搬进课堂。

## 4. Tiktokenizer —— Token 切分可视化（访问稳定性一般）

- 链接：https://tiktokenizer.vercel.app
- 演示内容：输入任意文字（支持中文），实时显示它被切成哪些 token、各 token 的编号。
- 课堂用法：配合第 5 页（Token）：现场输入学生说的句子，数一数被切成了几块，验证"token 不等于字也不等于词"。
- 注意：该站部署在 Vercel 上，国内访问有时不稳定，课前务必测试；打不开就用课件第 5 页的内置演示替代。

## 5. CS Unplugged —— 不插电活动方法论

- 链接：https://csunplugged.org/en/topics/artificial-intelligence/
- 内容："不插电计算机科学"官方网站的人工智能专题，汇集用游戏和角色扮演讲 AI 的活动设计，是本课程"人类 ChatGPT""黑箱预测机"等活动的同类方法论参考。
- 用法：教师备课、设计备用活动时参考。英文界面。
